# 프로젝트 개요
+ 개발배경
  - 영상 재생과 관련한 다양한 편의 기능을 만들어 손쉽고 편리한 영상시청을 가능하게 하고자 함

  - Visual Studio 2019를 이용


+ 플레이어의 기능
  - 영상 재생 기능
